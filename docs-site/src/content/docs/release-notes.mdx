For product releases and implementation history, see the [Overlord GitHub repository](https://github.com/cooperativ-labs/Overlord). For hosted service availability, use the status link provided by your deployment administrator.

When reporting a problem, include the Overlord version from `ovld version`, the backend mode, and the output of `ovld doctor` without sharing credentials.
