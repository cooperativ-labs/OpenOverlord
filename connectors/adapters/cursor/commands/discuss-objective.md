Submit a draft objective for discussion without starting execution.

Use the text after `/discuss-objective` as the ticket ID when provided.
If no ticket ID was provided, ask the user for one and stop.

Run:
`ovld protocol discuss-objective --ticket-id <ticketId>`

Rules:
- Use this when the user wants to open or discuss a ticket, not execute it.
- Do not attach after this command unless the user explicitly asks to execute.
