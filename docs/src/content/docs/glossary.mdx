---
title: Glossary
description: Short definitions for common Overlord terms.
---

- **Agent:** The coding tool that performs an objective.
- **Connector:** The integration that lets an agent attach, update, ask, and deliver.
- **Mission:** A durable record for a larger goal.
- **Objective:** One actionable step within a mission.
- **Project resource:** A repository, checkout, or other source associated with a project.
- **Execution target:** A machine or service where an objective runs.
- **Delivery:** The completed handoff from an agent.
